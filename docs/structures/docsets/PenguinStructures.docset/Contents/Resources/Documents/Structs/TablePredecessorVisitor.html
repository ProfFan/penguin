TablePredecessorVisitor

public struct TablePredecessorVisitor<Graph> where Graph : IncidenceGraph, Graph.VertexId : IdIndexable
extension TablePredecessorVisitor: TreeSearchVisitor, DFSVisitor, BFSVisitor
extension TablePredecessorVisitor: DijkstraVisitor

A graph algorithm visitor that records the parents of every discovered vertex.

PredecessorVisitor allows capturing a representation of the DFS tree, as this is often a useful output of a DFS traversal within other graph algorithms.
