Graphs

  • topologicalSort(_:)

    Computes a topological sort of graph.

    A topological sort means that for every 0 <= i < j < graph.vertexCount, there does not exist an edge from returnValue[j] to returnValuei.

    The implementation for topologicalSort is primarily a call to depthFirstTraversal.
