public enum Graphs

A namespace for graph-related functions.

  • depthFirstSearchNoInit(_:colorMap:visitor:start:)

    Runs depth first search on graph starting at startVertex using colorMap to keep track of visited verticies; visitor is called regularly to allow arbitrary state to be computed during search.

    Note

    graph is taken inout because the colorMap or visitor may store data within the graph itself.

    Precondition

    ColorMap has been initialized for every vertex to .white.
