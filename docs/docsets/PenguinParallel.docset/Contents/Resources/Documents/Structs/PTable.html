PTable

A collection of named PColumns.

A PTable, also known as a data frame, represents a tabular collection of data.

    Drops columns.

    This is the safe variation of drop(_:), which will throw an error if there is a problem with a provided column name.

    Note: this function is implemented such that it either fully succeeds or throws an error, and will never leave the Table in an inconsistent state.
